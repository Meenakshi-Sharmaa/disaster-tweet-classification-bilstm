# Disaster Tweet Classification using BiLSTM

## Project Overview

This project uses Natural Language Processing (NLP) and Deep Learning to classify tweets as disaster-related or non-disaster-related.

The model was built using a Bidirectional Long Short-Term Memory (BiLSTM) network and trained on the Disaster Tweets dataset.

---

## Dataset

Total Tweets: 7,613

Classes:

- 0 → Non-Disaster Tweet
- 1 → Disaster Tweet
